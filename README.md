# Zone-Transfer-Detector

It is simple tool that dectects zone transfer vulnerability.

# Working Way:

1.Domain

``` python3 detect.py -t/--target zonetransfer.me ```


2.Multiple Domain Wordlist


``` python3 detect.py -w/--wordlist <file-name>```
